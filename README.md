# UrbanStyle - Proyecto Final de Desarrollo Web

Este repositorio contiene el código fuente de una tienda online de ropa funcional. El proyecto fue desarrollado utilizando estándares web modernos y consume datos reales mediante una API pública.

## Descripción del Proyecto

El sitio web simula un e-commerce completo con las siguientes características:
- Carga dinámica de productos desde FakeStoreAPI.
- Filtrado de productos por categoría en tiempo real.
- Vista de detalle de producto individual (routing por parámetros URL).
- Diseño adaptable (Responsive Design) para dispositivos móviles y escritorio.
- Menú de navegación funcional y formulario de contacto.

## Tecnologías Utilizadas

- **HTML5:** Estructura semántica del sitio.
- **CSS3:** Diseño visual, uso de Variables CSS, Flexbox y CSS Grid Layout.
- **JavaScript (ES6):** Lógica de programación, Fetch API y manipulación del DOM.

## Estructura del Sitio

El proyecto consta de las siguientes vistas interconectadas:
1. index.html (Página de Inicio)
2. productos.html (Catálogo completo con filtros)
3. detalle.html (Información específica del producto)
4. blog.html (Sección de novedades)
5. nosotros.html (Información institucional)
6. contacto.html (Formulario de contacto)

## Instrucciones de Instalación

1. Clonar el repositorio:
   git clone https://github.com/AlvaroCosta04/Final.git

2. Ingresar a la carpeta del proyecto:
   cd Final

3. Ejecución:
   Puede abrir el archivo "index.html" directamente en su navegador web favorito o utilizar una extensión como "Live Server" para visualizar el proyecto.
